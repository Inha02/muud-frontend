## 🖥️ 프로젝트 소개

안녕하세요. 음악 감정 기록 서비스 ‘뮤드(MUUD)’입니다! <br> 
뮤드는 감정 테스트를 통해 감정별 플레이리스트를 추천받고, 플레이리스트와 함께 감정 일기를 기록할 수 있는 서비스입니다. <br>
사용자가 매개체로 자신의 감정을 이해하고, 꾸준히 상호작용하게 만드는 가치를 제공하고자 합니다.
